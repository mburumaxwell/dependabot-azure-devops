## Quick Start

1. Fork the repository
2. Run `pnpm install`
3. Make your changes
4. Run `pnpm build`
5. Run quality checks:

   ```bash
   pnpm lint
   pnpm test
   pnpm format:check
   ```

6. Submit a pull request
